# Countdown Timer Website

A simple full-stack countdown timer built using **React (Vite)** and **Node.js (Express)**.

## Features
- Set a countdown time in seconds.
- Displays remaining time live.
- Backend stores countdown state in memory (no database).
- Auto-updates every second.

## Folder Structure
```
countdown-timer-app/
├── backend/
├── frontend/
└── README.md
```

## Setup

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Open your browser at http://localhost:5173
